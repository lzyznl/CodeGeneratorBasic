### 使用说明

根据您的系统环境执行我们所提供给您的脚本

> generate <命令> <命令参数>

使用示例

> generate --needGit --a --o --l 

### 参数说明

（1） needGit

描述：是否需要生成.gitignore文件

类型：boolean

默认值：true

命令缩写：-needGit

（2） author

描述：作者注释

类型：String

默认值：lzy

命令缩写：-a

（3） outputText

描述：输出信息

类型：String

默认值：result:

命令缩写：-o

（4） loop

描述：是否开启循环

类型：boolean

默认值：false

命令缩写：-l
